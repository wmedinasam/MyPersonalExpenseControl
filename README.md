# My Personal Expense Control

Una aplicación diseñada para ayudarte a gestionar y controlar tus gastos personales de manera sencilla y eficiente.

## 📌 Descripción

My Personal Expense Control es una herramienta que permite registrar, categorizar y analizar tus gastos diarios. El objetivo es proporcionar una visión clara de tus hábitos de gasto y ayudarte a tomar decisiones financieras más informadas.

## 🛠️ Tecnologías Utilizadas

- **Frontend**: HTML, CSS, JavaScript / React
- **Backend**: ASP.NET Core / Node.js
- **Base de Datos**: SQL Server / MongoDB
- **Control de Versiones**: Git
